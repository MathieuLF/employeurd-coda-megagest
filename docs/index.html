Utilitaire Windows

Créer un MND à partir d'une écriture EmployeurD.

Un outil simple pour préparer l'import de paie dans MégaGest, avec validation des soldes avant la création du fichier final.

Télécharger la dernière version Demander du support

En pratique

Trois étapes.

1. Ajouter le TXT

Choisir l'écriture détaillée standard de EmployeurD.

2. Confirmer les totaux

Ajouter le SPD640-P au besoin pour corroborer les soldes.

3. Créer le MND

Générer le fichier seulement lorsque les validations sont conformes.

Sécurité

Les fichiers de paie restent sur le poste.

L'application ne transmet pas les fichiers EmployeurD, SPD640-P, MND, Markdown ou JSON. Les versions officielles sont publiées avec des empreintes SHA256 et un rapport VirusTotal de l'exécutable.

Les exemples du dépôt sont synthétiques. Aucun fichier réel ne doit être joint à un billet public.
